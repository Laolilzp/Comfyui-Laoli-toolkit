# Comfyui-Laoli-toolkit
Comfyui-Laoli-toolkit 常用小工具合集


1、老李-可视化Sigma编辑器 —— 单节点包含调度器加载、步数、降噪设置、曲线调节，可实现曲线到数据、数据到曲线双向调节，并可保存为具体设定，一键调取。使用时初次先点一下右上角运行，获取某模型的默认sigma后再进行调整。

2、音频自动切分&文本合并去重 —— 配合Qwen-ASR插件使用，主要作用是将长音频自动切分以减少爆显存和提高识别效率，同时包含文本合并去重节点和音频自动切分节点。具体详情说明后续再更新。
